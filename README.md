## Product Requirements

- This is a platform where users create products.
- Sorting by any column (ascending and descending) by clicking the column header
- Search by product name or SKU
- Filter by category using a dropdown
- Pagination — 10 rows per page, configurable
- Visual highlight on rows where stock quantity is at or below the low stock threshold
- Inline Stock Editing

## Run Locally

Clone the project

```bash
  git clone https://github.com/2rohith2/inventory-api.git
```

Go to the project directory

```bash
  cd inventory-api
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Link to access

```bash
  http://localhost:8080/api/v1/products
```

To build & run

```bash
  npm run build
  node /dist/bin/www.js
```
